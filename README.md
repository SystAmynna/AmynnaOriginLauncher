# Amynna's Origin Launcher

Ce projet est un lanceur personnalisé pour le client (Minecraft moddé) du serveur OriginRP.

## ROADMAP  

- Configs
    - Paramètrer la RAM (minimum et maximum)
    - Paramètrer la résolution de la fenêtre
    - Mode d'affichage patate (pour les petits PC)
    - Connexion directe au serveur OriginRP
    - Afficher (on / off) les news du serveur
    - Reconnexion automatique (à Mojang / Microsoft)
    - Vérifier les fichiers du jeu (intégrité)
    - Supprimer / Désinstaller le Launcher
- Menu Mods optionnels
- Mode Administrateur (pour les techniciens du serveur)
- Interface Graphique améliorée
